Introduction to abcsh

This is a web page about abcsh. abcsh is a shell for processing ABC music notation content. The ABC music notation is a popular simple ASCII music notation system used by many internet music enthusist. You can find out more information about ABC music notation at the ABC Home Page.

Downloading abcsh

Since abcsh is still in prototype mode, there is no compiled version available. If you would like to browse the CVS source for abcsh take a look at abcsh CVS repository.

Documentation

Draft abcsh documentation can be found at http://www.gnu.org/software/abcsh/documentation/.
